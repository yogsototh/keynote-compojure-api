Ring: simple concepts

  • Handlers: Request → Response
  • Middlewares: Handler → Handler
  • Adapters: HTTP req → Handler → HTTP resp

Ring basic job of handler: routing

(cond
  (= (:uri request) "/") handler-one
  (= (:uri request) "/pizza") handler-two)

? GET /pizza/cheese?olive=8 ⇒
        (handler-get-pizza pepperoni {:olive 8})

? POST /pizza {:name "pepperoni" :ingredients ["olive"]} ⇒
        (handler-post-pizza pepperoni )

Compojure: routing

(GET "/pizza/:id" (handle id))
